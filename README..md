# Emergency Room Trends 🏥📊

**Objective:**  
Explore trends in emergency room visits by time, type, and demographics. Predict wait times using simple regression.

**Tools Used:**  
- Python (Pandas, Matplotlib, Seaborn)  
- Jupyter Notebook  
- Linear Regression

**Skills Demonstrated:**  
- Data wrangling  
- Time-series trend analysis  
- Predictive modeling  
- Public health insights

**Dataset:**  
[Ontario Emergency Department Wait Times](https://data.ontario.ca/en/dataset/emergency-department-wait-times)

📈 This project reflects my interest in healthcare systems, efficiency, and using data to improve patient outcomes.
